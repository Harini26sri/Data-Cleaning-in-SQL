# Data-Cleaning-in-SQL

**MySQL Data Cleaning Project**

Hands-on project demonstrating essential data cleaning techniques using **MySQL** on real-world messy datasets. Focuses on transforming raw, inconsistent data into analysis-ready tables through SQL queries.

## Key Techniques Implemented
- **Duplicate removal** using `ROW_NUMBER()` and CTEs
- **Missing value handling** with `COALESCE()` and conditional `UPDATE`
- **Data standardization** (date formats, text casing, categorical mapping)
- **Outlier detection** and trimming with window functions
- **Data validation** using constraints and `CHECK` clauses

## Technologies
```
MySQL 8.0+ | SQL Queries | Window Functions | CTEs
```
